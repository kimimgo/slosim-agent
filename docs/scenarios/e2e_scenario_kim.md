# E2E 시나리오: 김 대리의 첫 슬로싱 해석

> 조선소 구조설계팀 김 대리(29). CFD 경험 없음. 팀장이 "LNG 화물창 슬로싱 검토 좀 해봐"라고 던졌다.

---

## Act 1. 첫 만남 — 자연어 입력 + AI 조건 추론

김 대리가 터미널에서 `./slosim-agent`를 실행한다. BubbleTea TUI가 뜬다.

```
김 대리: LNG 화물창 슬로싱 해석 좀 해줘.
         탱크 크기는 가로 40m 세로 30m 높이 27m이고
         물 60% 채워져 있어. 배가 좌우로 0.12Hz로 흔들려.
```

AI가 **조건을 추론**하고 확인을 요청한다:

```
AI: 입력하신 조건을 정리했습니다.

    ┌─ 해석 조건 요약 ─────────────────────┐
    │ 탱크    : 40 × 30 × 27 m (직사각형)  │
    │ 유체    : 물, 높이 16.2m (60%)        │
    │ 가진    : 0.12 Hz, 진폭 5°(롤)       │
    │ 정밀도  : 입자 간격 0.15m             │
    │ 해석시간: 30초 (약 3.6주기)           │
    │ 예상 파티클: 약 480만개               │
    └──────────────────────────────────────┘

    ⚠ 공진 분석: 1차 공진 주파수 0.098Hz,
      주파수비 1.22 → "공진 근처" 입니다.
      수위 변동이 클 수 있으니 유의하세요.

    이 조건으로 진행할까요?
```

**커버 기능**: `SloshingCoderPrompt`, `xml_generator`
