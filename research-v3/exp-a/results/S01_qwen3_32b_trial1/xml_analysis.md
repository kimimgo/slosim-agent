# S01 XML Parameter Fidelity Analysis (Preliminary)

## 프롬프트
SPHERIC 벤치마크 탱크(900x62x508mm)에 해바라기 오일을 18% 채워서
슬로싱 해석해줘. 오일 밀도 990, 점도 0.045 Pa·s.
주기 1.535초로 바닥 중심 기준 피치 회전 4도.

## XML 분석 (simulations/sloshing_case.xml)

| 파라미터 | 요청값 | XML 값 | 판정 |
|---------|--------|--------|------|
| 탱크 크기 | 900×62×508mm | 0.9×0.062×0.508m | PASS |
| 유체 밀도 | 990 kg/m³ (오일) | 1000 kg/m³ (물) | **FAIL** |
| 점도 | 0.045 Pa·s | Visco=0.01 (기본값) | **FAIL** |
| 유체 높이 18% | 91.44mm | 91.44mm | PASS |
| 여진 주파수 | 1/1.535=0.6515Hz | 0.6515 Hz | PASS |
| 여진 타입 | 피치 회전 (mvrotsinu) | **병진 (mvrectsinu)** | **FAIL** |
| 여진 진폭 | 4도 | 0.0314m (병진) | **FAIL** |
| TimeMax | ~7s | 7.68s (5주기) | PASS |

## M-A3 결과: FAIL (4/8 PASS)

## 주요 오류
1. **여진 타입 혼동**: 피치 회전(mvrotsinu)을 병진(mvrectsinu)으로 생성
   - 0.0314m ≈ 4° × π/180 × 0.45m(탱크 길이 절반) → 등가 진폭으로 변환 시도한 것으로 추정
   - 그러나 물리적으로 완전히 다른 운동
2. **유체 물성 무시**: rhop0=1000(물 기본값), Visco=0.01(물 기본값)
   - 오일 밀도 990, 점도 0.045 Pa·s가 반영되지 않음
   - 도메인 프롬프트에서 rhop0 설정 방법이 부족할 수 있음
