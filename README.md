# slosim-agent

AI-powered sloshing simulation agent built on OpenCode with DualSPHysics SPH solver.

## Features
- Natural language to sloshing simulation setup
- DualSPHysics v5.4 GPU solver integration  
- Automated case generation, execution, post-processing
- AI-driven parameter selection with physical reasoning

## Requirements
- NVIDIA GPU (RTX 4090)
- Docker + NVIDIA Container Toolkit
- Ollama with Qwen3 model
