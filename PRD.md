# slosim-agent PRD (Product Requirements Document)

*v1.0 — 2026-02-13*

---

## 1. 개요

**프로젝트명:** slosim-agent
**목적:** CFD 비전문 실무자가 자연어로 슬로싱 해석을 수행할 수 있는 도메인 특화 AI 에이전트
**핵심 기여:** 에이전트 아키텍처 + 현장 실무자가 CFD 지식 없이 슬로싱 해석 수행 가능

## 2. 사용자

| 구분 | 설명 |
|------|------|
| Primary | 현장 운영 담당 실무자 (CFD 비전문가) |
| Secondary | 연구자 (마스터 본인) |

## 3. 인터페이스

- **형태:** TUI 확장 (BubbleTea 커스텀, OpenCode 기반)
- **디자인:** 슬로싱 해석 도메인 특화 아름다운 UI/UX
- **테스트 LLM:** Qwen3 (Ollama 로컬, 32b/8b)

## 4. 핵심 워크플로우

```
사용자 자연어 입력 (+ 선택적 STL 파일)
  ↓
[AI 해석] 탱크 형상, 유체 물성, 가진 조건, BC 추론
  ↓
[GenCase] XML 케이스 자동 생성 → 파티클 지오메트리
  ↓
[DualSPHysics] GPU SPH 시뮬레이션 (백그라운드 Job)
  ↓
[모니터링] 진행률, 수렴, 에러 실시간 감지
  ↓
[pvpython] ParaView CLI 후처리 → 이미지 + 애니메이션
  ↓
[AI 리포트] 해석 조건 요약 + 결과 시각화 + 물리적 해석
  ↓
사용자에게 결과 제시
```

## 5. 기능 요구사항

### 5.1 입력

| ID | 기능 | 우선순위 |
|----|------|----------|
| IN-01 | 자연어 입력 → AI가 해석 조건 추론 | P0 |
| IN-02 | STL 파일 직접 입력 (CAD 지오메트리) | P0 |
| IN-03 | 단순 입력 지원 ("LNG 탱크 슬로싱 해석" → AI가 표준 치수 제안) | P0 |

| ID | 기능 | 우선순위 |
|----|------|----------|

| SIM-02 | GenCase XML 자동 생성 | P0 |
| SIM-03 | AI 자율 판단: dp, 시뮬레이션 시간, 타임스텝, BC | P0 |

## 6. 기술 스택

| 레이어 | 기술 |
|--------|------|
| Agent Core | Go + BubbleTea TUI (OpenCode fork) |
| LLM | Qwen3 via Ollama (로컬) |
| Solver | DualSPHysics v5.4 (CUDA 12.6, RTX 4090) |
| Post-processing | pvpython (ParaView CLI) |
| Container | Docker + NVIDIA Container Toolkit |
| Report | Markdown |

## 8. 비기능 요구사항

| 항목 | 요구사항 |
|------|----------|
| 성능 | GPU 시뮬레이션, 백그라운드 Job으로 TUI 블로킹 없음 |
| 확장성 | 새 케이스 템플릿 추가 용이 |
| 에러 처리 | 시뮬레이션 발산/에러 자동 감지 + 사용자 안내 |
| 데이터 보존 | 모든 시뮬레이션 결과 + 리포트 이력 관리 |
